# Images des Projets

Ce dossier contient toutes les images utilisées pour les projets du portfolio.

## Structure recommandée :

```
projects/
├── ctmhb-handball-management/
│   ├── logo.png
│   ├── screenshot-1.jpg
│   └── screenshot-2.jpg
├── fitness-tracker/
│   ├── app-icon.png
│   └── screenshots/
└── gym-management/
    ├── logo.png
    └── dashboard.jpg
```

## Utilisation :

Pour utiliser une image dans un projet, utilisez le chemin :
```typescript
image: "/images/projects/nom-du-projet/nom-de-l-image.jpg"
```

## Formats recommandés :
- **Logos** : PNG avec transparence
- **Screenshots** : JPG ou PNG
- **Dimensions** : 800x600px minimum pour les cartes de projets
- **Optimisation** : Compresser les images pour le web
